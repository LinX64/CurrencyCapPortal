Currency Cap Portal

Free Currency & Crypto API

Endpoints

GET /latest.json

Current Iranian currency exchange rates (Bonbast)

https://linx64.github.io/CurrencyCapPortal/latest.json

GET /history/1d.json

Yesterday's rates (Hansha → Bonbast fallback)

https://linx64.github.io/CurrencyCapPortal/history/1d.json

GET /history/1w.json

Last week's rates (Hansha → Bonbast fallback)

https://linx64.github.io/CurrencyCapPortal/history/1w.json

GET /history/1m.json

Last month's rates (Hansha → Bonbast fallback)

https://linx64.github.io/CurrencyCapPortal/history/1m.json

GET /history/1y.json

Last year's rates (Hansha → Bonbast fallback)

https://linx64.github.io/CurrencyCapPortal/history/1y.json

GET /crypto.json

Top 100 cryptocurrencies by market cap

https://linx64.github.io/CurrencyCapPortal/crypto.json

GET /news.json

Latest blockchain news articles

https://linx64.github.io/CurrencyCapPortal/news.json

Usage Examples

Latest Rates

const rates = await fetch('https://linx64.github.io/CurrencyCapPortal/latest.json')
  .then(res => res.json());

Period-Based History (Mobile-Friendly!)

// Get last week - perfect for mobile apps!
const weekData = await fetch('https://linx64.github.io/CurrencyCapPortal/history/1w.json')
  .then(res => res.json());

// Get last month for charts
const monthData = await fetch('https://linx64.github.io/CurrencyCapPortal/history/1m.json')
  .then(res => res.json());

// Get last year for trends
const yearData = await fetch('https://linx64.github.io/CurrencyCapPortal/history/1y.json')
  .then(res => res.json());

// Available periods: 1d, 1w, 1m, 1y
